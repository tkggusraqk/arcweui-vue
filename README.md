# arcweui-vue
weui的vue版本
# 说明
样式依赖weui css，[链接地址](https://github.com/Tencent/weui/wiki/getting-started)

#安装/使用

   npm install arcweui-vue --save

   main.js 引入依赖：
``` javascript
import ArcWeUI from 'arcweui-vue'
import 'arcweui-vue/dist/arcweui-vue.min.css'
Vue.use(ArcWeUI)
```
   具体使用请到github 下载项目看examples
